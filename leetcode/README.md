# LeetCode Solutions in Python

This directory contains solutions to popular LeetCode problems, focusing on beginner-friendly challenges that help build fundamental programming skills.

## Problems Included

1. **Two Sum** (Easy) - Find two numbers that add up to a target
2. **Palindrome Number** (Easy) - Check if a number is a palindrome
3. **Roman to Integer** (Easy) - Convert Roman numerals to integers
4. **Reverse String** (Easy) - Reverse a string in-place
5. **FizzBuzz** (Easy) - Classic programming challenge

## How to Use

Each solution file contains:
- Problem description
- Solution approach explanation
- Python implementation with comments
- Time and space complexity analysis
- Example test cases

## Running the Solutions

You can run any solution file directly:

```bash
python leetcode/two_sum.py
python leetcode/palindrome_number.py
python leetcode/roman_to_integer.py
python leetcode/reverse_string.py
python leetcode/fizzbuzz.py
```

## Learning Tips

- Try to solve each problem on your own first
- Understand the time and space complexity
- Practice different approaches to the same problem
- Focus on writing clean, readable code with comments
